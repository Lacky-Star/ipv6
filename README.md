curl食用用方法 #前提得装curl 如果没安装  Ubuntu和Debian 使用 apt install curl -y   centos和rocky 则使用 yum install curl -y


bash <(curl -sL https://raw.githubusercontent.com/Lacky-Star/ipv6/main/ipv6.sh)

wget食用用方法 #前提得装wget 如果没安装  Ubuntu和Debian 使用 apt install wget -y   centos和rocky 则使用 yum install wget -y



bash <(wget -qO- https://raw.githubusercontent.com/Lacky-Star/ipv6/main/ipv6.sh)
